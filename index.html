Services

What Can I Do For You?

Since beginning my journey as a software engineer over 5 years, I've done both on-site and remote work for companies, consulted for startups, and collaborated with talented people to create digital products for both business and consumer use. I'm quietly confident, naturally curious, and perpetually working on improving my chops one design problem at a time.

Mobile App Development

With over 3 years of experience in React Native development, I have developed and deployed many mobile applications to the playstore and app store, and secured servers for both global and internal users. I will help you to turn your idea into a hybrid mobile application with the best mobile architecture and coding standards for both Android and iOS.

Web Development

I have a track record of delivering products on time, without defects, and in sprints. With my experience in React JS, HTML, SCSS, and JavaScript, I delivered 4 enterprise webapps, 16 websites, 7 webpages, and online portfolios.

Architecture Setup

Help clients and other freelancing developers with the initial project set-up for front-end applications with Redux for global state management; Redux-Saga and Redux-Thunk for middleware to interact with backend services; React Router Dom for routing within the web apps and websites; React Navigation for mobile app navigation; multi-language support for both mobile and web apps; and global toast messaging.

User Testing

Support the product by performing thorough testing of all the functional and UI flows of the application by mapping against the product requirements, tracking the issues over JIRA tickets, and creating a proof document to support the test scenarios and test cases.

Data Analytics

Deduce logical analysis by performing various analytical techniques on a raw data set. Perform test data cleaning and data splitting for training, testing, and validation; explore overoutliers and boundary values; prescribing new actions to improve the productivity of an existing process.

Defect Fixes

Assist clients in identifying the root cause of bugs, providing fixes, improving the performance and security of existing applications, and implementing best code practises to make it easier for future developers to debug issues.
